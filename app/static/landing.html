ICAR ADE v1.5 API

REST API for ICAR Animal Data Exchange ADE v1.5 using FastAPI and TinyDB

📘 Swagger UI 📄 ReDoc 💚 Healthcheck GitHub

Resources

🐄 Animals

CRUD for animal resources: create, list, get, update, delete animals with ICAR ADE payloads.

icarAnimalCoreResource

📅 Events

CRUD for events linked to animals: weight, treatment, reproduction, health, movement, milking, etc.

20+ event types

👥 Groups

CRUD for animal groups and group-level events (feedings, movements, treatments, weights).

icarAnimalSetResource

📡 Devices

CRUD for devices: sensors, collars, ear-tag readers and other hardware identifiers.

icarDeviceResource

💊 Medicines

CRUD for medicines and pharmaceutical products used in treatments.

icarMedicineResource

📍 Locations

CRUD for locations: farms, barns, pens, feedlots and other physical locations.

icarLocationResource

API Routes

  • GET/resourcesList registered resource types
  • POST/resourcesCreate any resource type generically
  • GET/resources/{id}Get any resource by ID
  • GET/animalsList animals
  • POST/animalsCreate animal
  • GET/animals/{id}Get animal by ID
  • PUT/animals/{id}Update animal
  • DELETE/animals/{id}Delete animal
  • POST/eventsCreate an event for an animal
  • GET/events?animalId={id}List events for an animal
  • GET/events/{id}Get event by ID
  • GET/healthcheckAPI health status

Supported Events

The API supports 20+ ICAR ADE event types for animal tracking:

⚖️ WeightEvent

Individual animal weight observations.

💉 TreatmentEvent

Individual treatments and medication administration.

💊 MedicineTransaction

Medicine usage tracking per animal.

🫃 Repro. events

Heat, insemination, pregnancy check, parturition, abortion, etc.

🚚 Movement events

Arrival, departure, birth, death.

🥛 Milking events

Milking visits, dry-off, daily averages, test-day results.

🩺 Health events

Diagnosis, health status observation, attention.

🐑 Group events

Group feeding, weight, treatment, movement, position observation.
